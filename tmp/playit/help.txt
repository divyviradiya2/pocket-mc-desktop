Usage: playit.exe [OPTIONS] [COMMAND]

Commands:
  version      
  account      
  claim        Setting up a new playit agent
  start        Start the playit agent
  tunnels      Manage tunnels
  reset        removes the secret key on your system so the playit agent can be re-claimed
  secret-path  shows the file path where the playit secret can be found
  help         Print this message or the help of the given subcommand(s)

Options:
      --secret <SECRET>     secret code for the agent
      --secret_path <PATH>  path to file containing secret
  -w, --secret_wait         wait for secret_path file to read secret
  -s, --stdout              prints logs to stdout
  -l, --log_path <PATH>     path to write logs to
      --platform_docker     overrides platform in version to be docker
  -h, --help                Print help
